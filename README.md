# USDA NASS Data Connector

Fetches agricultural statistics from USDA National Agricultural Statistics Service QuickStats API.

## Data Sources

- **QuickStats API**: https://quickstats.nass.usda.gov/api
- Requires API key (set NASS_API_KEY environment variable)
- Max 50,000 records per request

## Datasets

- Crop production and yields
- Livestock counts and production
- Prices received by farmers
- Agricultural economics
- Census of Agriculture data
